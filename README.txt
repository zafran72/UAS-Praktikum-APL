Studi kasus: Tree( Dari huruf T )
Ini aplikasi cli di mana user dapat memilih opsi tree yang ingin diambil. 
Namun selain memilih tree, user juga dapat memilih agar 
tree tersebut dapat diadaptasi dalam format fruit.

Pattern yang digunakan:
> Strategy Pattern
    - Terdapat pada package Tree
    - Diterapkan dalam memilih strategy tree mana yang ingin ditonton

> Adapter Pattern
    - Terdapat pada package TreeAdapter
    - Diterapkan dalam mengadaptasi tree ke dalam format fruit
